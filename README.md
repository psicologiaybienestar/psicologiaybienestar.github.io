# Psicología & Bienestar

Aplicación híbrida para el centro de psicología y bienestar emocional, desarrollada con **Ionic + Angular + Supabase**. Disponible como app Android nativa y versión web progresiva.

## Acerca de

Plataforma integral que ofrece recursos de bienestar emocional, gestión de citas, contenido psicoeducativo y un espacio seguro para el acompañamiento terapéutico. Tanto pacientes como administradores acceden a herramientas diseñadas para facilitar la comunicación y el seguimiento.

### Para usuarios
- Landing page informativa con servicios, equipo y testimonios
- Noticias y eventos del centro
- Galería de imágenes
- Solicitud de citas en línea
- Frases motivacionales diarias, consejos emocionales y actividades de bienestar
- Check-in emocional con seguimiento semanal
- Minijuegos terapéuticos (memoria, respiración, gratitud, reacción, quiz emocional)
- Notificaciones push recordatorias (3 al día: frase, actividad, consejo)

### Para administradores (web)
- Dashboard con estadísticas
- CRUD completo de noticias, eventos, frases, consejos, actividades, emociones, minijuegos
- Gestión y moderación de testimonios
- Galería con carga drag & drop
- Importación/exportación masiva via XLSX
- Notificaciones push automáticas al publicar contenido nuevo

## Tecnologías

**Ionic Angular** — UI híbrida y navegación móvil · **Angular 20** — standalone components · **Supabase** — Auth, base de datos, almacenamiento, tiempo real · **Capacitor** — Android nativo · **TailwindCSS v3** · **TipTap** — editor de texto enriquecido · **Swiper** — sliders · **Google Sheets** — fuente de testimonios · **Formspree** — formulario de contacto · **Firebase Cloud Messaging** — notificaciones push · **Lucide** — iconos
